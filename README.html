    (quote ((sequence "TODO(t)" "NEXT(n)" "|" "DONE(d)")
            (sequence "WAITING(w@/!)" "HOLD(h@/!)" "|" "CANCELLED(c@/!)"))))

1.2. Cute characters instead of asterisk:

This Snippet This is the cute starts that are positioned before a heading (default they all are asterisks) 

;; Installed with M-x package-install RET org-bullets RET
(use-package org-bullets
      :ensure t
      :config
      ;; Tbh It's pretty random now. But I can change it later
      (setq org-bullets-bullet-list '("▶" "◉" "§" "Σ" "Ψ" "Ω" "β" "δ" "ᗧ" "᠉" "⋑" "✸")))
